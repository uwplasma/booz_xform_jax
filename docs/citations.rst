Suggested Citation Practice
---------------------------

If you publish results produced with ``booz_xform_jax``, cite:

- the original Boozer-coordinate paper,
- the BOOZ_XFORM application paper,
- the software repository for ``booz_xform_jax``,
- any upstream equilibrium or optimization packages used in the workflow.
